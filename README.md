# DNN Pose Estimator in GoLang

Simple **pose estimator** written in GoLang on top of GoCV (Open Computer Vision porting for Go).

# Download Coffee Model

```sh
$ sh ./install.sh
```

# Build:

```sh
$ go build main.go
```

# Run:

```sh
$ ./main
```
